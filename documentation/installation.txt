# Installation
- Installer PHP / MySQL
- Créer un répetoire RGPD dans www ou htdocs, selon le serveur web
- Copier dans ce répertoire toute l'arborescence de RGPD
- Créer la base de donnée exécuter :
	- GSS\sql\Creation_base_RGPD.sql -> crée une base de données nommée "rgpd"
	- GSS\sql\Grant_User_RGPD.sql -> crée un utilisateur "rgpd", mot de passe "sql_rgpd" ayant les droits SELECT, INSERT, UPDATE, DELETE sur la base nommée "rgpd" (modifier éventuellement les 2 "localhost" de ce script pour les remplacer par le serveur qui accèdera à la base (dans le cas ou le serveur PHP est différent du serveur MySQL))
- Modifier le fichier "configuration.ini" :
	- Utilisation du LDAP ou non
	- Host de la BDD
- Lancer avec http://host:port/RGPD/
